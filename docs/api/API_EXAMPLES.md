## Use Cases for Authentication
 - All employees have an employee number, so a list of administrators can be made, and if their employee number is apart of this list, they will have access to features specific to the administrator
 - Similarly with the receptionist, if their employee numbers are in a list describing their specific position, then they will be given access to features specific to their qualifications

## Use Cases for Room Management
 - Facility managers will be able to make a room unavailable for maintenance
 - Administrators will be able to control room booking and view room availability

## Use Cases for Booking
 - Employees will be able to view a list of rooms available for booking
 - Employees will be able to filter the list of rooms by the specific capacity required for the meeting
 - Employees will be able to cancel/delete an existing booking, given that they were the person who made the booking
